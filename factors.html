There are certain factors which every programmer in the world have consider whether its for a client or for personal use. These factors are:

Organisational Policy

In any working organization there are certain rules and regulations which have to be followed when using hardware and software. For example, when you are picking a programming language to create software that can only be used on a windows 10 software and hardware the programing language that has been chosen has to support windows 10 e.g. C# and C++.

Suitability in terms of available features and tools

During the stage of creating software a suitable language will need to be chosen to suit the needs of the program. For example, if you are making new hotel app you may use C# as it has a variety of features and tools such as debugging and editing tools.

Availability of trained staff

In any organization there has to be trained and available staff at all times so that they can deal with any software programming issues which the user is finding difficult to solve. For example If the programmer is working on a project where they have to create system and the organisation needs to recruit the right person , the organisation will employ a person who is more familiar with the language as they will be considered as experienced software designers in the used language by the programmer.
